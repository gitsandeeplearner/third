There are following properties of display: none, block, inline, inline-block
display: none -> Yeh us div ko hide kar dega jis per yeh laga hua hai .
display: block -> Yeh new line se shuru hota hai aur ismein hum width, height, margin, padding de sakte hain. By default yeh puri width leta hai.
display: inline -> Yeh span ki tarah kam karta hai. Yeh utni hi jagah leta hai jitni isko jarurat hoti hai. Yeh new line se shuru NAHI hota hai aur upper wala tag khatam hote hi shuru ho jata hai. Ismein width, height, padding, margin kam nahi karte hain.
display: inline-block -> Yeh bi EK hi line se shuru hota hai aur width, height, margin, padding sabi kam karte hain.
display:flex: PARENT div ko display flex do to INTERNAL wale sare div's/block's inline ho jayege.
